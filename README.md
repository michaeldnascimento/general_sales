# Docker | PHP 7.4 | Mysql 5.7 | Phpmyadmin


#### 1. Run Docker

Start Server (in background)

- <b>docker-compose up -d</b>

Stop Server

- <b>docker-compose down</b>

Rebuild Server

- <b>docker-compose up -d --build</b>
